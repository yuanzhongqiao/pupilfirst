<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">反思自己的工作并做出改进。</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">最后，通过回答同伴的问题来参与学习过程。</font></font></li>
